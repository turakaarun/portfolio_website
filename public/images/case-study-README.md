# Case Study Images

Place your case study images here.

## Image Naming Convention:

For Project 1 (case-study/1):
- `case-study-1-1.jpg` - First image
- `case-study-1-2.jpg` - Second image
- `case-study-1-3.jpg` - Third image
- `case-study-1-4.jpg` - Fourth image

For Project 2 (case-study/2):
- `case-study-2-1.jpg` - First image
- `case-study-2-2.jpg` - Second image
- `case-study-2-3.jpg` - Third image
- `case-study-2-4.jpg` - Fourth image

## Recommended Specifications:

- **Size**: 1920x1080px (16:9 aspect ratio)
- **Format**: JPG, PNG, or WebP
- **Optimize**: Compress images for web performance
- **Quality**: High quality but optimized file size

## How It Works:

The case study pages will automatically display these images when you add them to this folder. The images are referenced in `src/pages/CaseStudy.jsx`.
