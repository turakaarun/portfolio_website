# Images Folder

Place your project images here.

## Recommended Image Sizes:

- **Project Images**: 1920x1080px (16:9 aspect ratio)
- **Motion Design Samples**: 1920x1080px (16:9 aspect ratio)
- **Profile Photo** (optional): 800x800px (square)

## File Naming:

- `project1.jpg` - First project image
- `project2.jpg` - Second project image
- `motion-sample1.gif` - Motion design sample
- `profile.jpg` - Profile photo (optional)

## Supported Formats:

- JPG/JPEG
- PNG
- GIF (for animations)
- WebP (recommended for better performance)

## How to Use:

1. Add your images to this folder
2. Update the image paths in `src/components/Projects.jsx`
3. The images will automatically display in your portfolio
